================================
CSV 批量翻译工具 - 使用说明
================================

感谢使用本工具！这是一个可以批量翻译 CSV 文件的小程序。

-----------------
## 快速开始
-----------------

1.  **运行程序**:
    双击 `CSVTranslator.exe` 来启动程序。
    程序启动后，会自动创建 `input` 和 `output` 两个文件夹（如果它们不存在的话）。

2.  **设置 API Key**:
    - 打开程序后，在界面上方的输入框里，粘贴你的 DeepL API Key。
    - 点击【保存 API Key】按钮，这样下次就不用重新输入了。
    - 强烈建议点击【测试 API Key】来确认你的 Key 是不是有效的。

3.  **放入文件**:
    将你想要翻译的 CSV 文件，全部复制或移动到 `input` 文件夹里。

4.  **开始翻译**:
    - 在程序主界面，点击大大的【开始批量翻译】按钮。
    - 程序会开始处理 `input` 文件夹里的所有 CSV 文件，你可以在下方的日志区看到实时进度。

5.  **获取结果**:
    - 翻译完成后，日志区会提示“全部处理完成”。
    - 这时，你可以去 `output` 文件夹里找到已经翻译好的文件，文件名和原文件保持一致。

-----------------
## 注意事项
-----------------

- **CSV 文件要求**:
  - 你的 CSV 文件里必须有一个名为 `English(en)` 的列，作为翻译的源语言。
  - 其他需要翻译的语言列，其列名也需要符合特定格式 (例如 `German(de)`, `Japanese(ja)` 等)。

- **关于覆盖**:
  - 默认情况下，本工具只会翻译目标语言列中的【空白单元格】。
  - 如果你想让工具重新翻译所有内容（包括已经有译文的），可以勾选界面上的“覆盖已存在的目标语言内容”选项。

- **遇到问题?**:
  - 如果提示 API Key 无效，请检查你的 Key 是否正确，或者网络是否通畅。
  - 如果文件没有被翻译，请检查 `input` 文件夹里是否有文件，以及 CSV 的列名是否正确。
